#Create an Editable Psychrometric Chart (SVG)

建築プレゼンテーションのために設計された、編集可能な空気線図生成ツールです。
EPW気象データをもとに、年間の気候傾向を密度として可視化した空気線図をSVG形式で出力します。

IllustratorやInkscapeでの編集を前提とし、「伝わる図」としての使いやすさに焦点を当てています。

特徴

密度ベースのプロットにより、点の重なりを分布として捉え、年間の気候傾向を直感的に把握できます。
不要な情報を省き、プレゼンテーションでの視認性を優先した構成です。
SVG形式で出力されるため、色や線、注釈などを自由に調整できます。
シンプルな操作で図を生成できるよう設計されています。

使い方
リポジトリをダウンロード
pythonをPCにダウンロード
run を実行
（これ以下はGUIで操作出来ます）
EPWデータを読み込む
make ボタンを押す
EPWデータの取得

EPWデータは以下からダウンロードできます。
https://www.ladybug.tools/epwmap/

このツールについて

一般的な環境解析ツールでは、点プロットによって情報が分散しやすく、そのままではプレゼンテーションに適さない場合があります。

このツールは、気候データを密度として整理することで、年間の傾向を一枚の図として把握できるようにし、さらに編集可能な形式で出力することで、設計資料への展開を容易にします。
